# 임사부의 오늘 식당

모바일에서 URL만 열면 바로 플레이할 수 있는 외식업 경영 웹게임입니다.

플레이어는 작은 식당 사장님 `임사부`가 되어 하루 동안 8번의 영업 상황을 선택합니다. 선택에 따라 매출, 고객 만족도, 재고, 리뷰 평점이 바뀌고 마지막에 하루 성적표가 나옵니다.

## 주요 기능

- 설치 없이 브라우저에서 바로 실행되는 정적 웹게임
- 모바일 세로 화면에 맞춘 단일 화면 진행
- 점심 러시, 재료 관리, 직원 운영, 날씨 변수, 리뷰 대응 등 외식업 상황 카드
- 선택지별 매출, 만족도, 재고, 리뷰 변화
- 영업 종료 후 S~D 등급과 운영 팁 제공

## 파일 구성

- `index.html`: 게임 화면 구조
- `styles.css`: 모바일 UI와 식당/주방 일러스트
- `app.js`: 게임 진행, 선택지, 점수 계산
- `.nojekyll`: GitHub Pages 배포용 설정

## 실행 방법

`index.html` 파일을 브라우저로 열면 바로 사용할 수 있습니다.

## URL로 배포하는 방법

GitHub Pages 같은 정적 호스팅에 이 폴더의 파일을 올리면 별도 서버 없이 URL 하나로 접속할 수 있습니다.

1. GitHub에서 새 저장소를 만듭니다.
2. 이 폴더의 파일을 저장소에 올립니다.
3. 저장소 `Settings`에서 `Pages`를 엽니다.
4. `Source`를 `Deploy from a branch`로 선택합니다.
5. `Branch`를 `main`, 폴더를 `/(root)`로 선택하고 저장합니다.
6. 잠시 뒤 아래 형식의 주소로 접속합니다.

```text
https://깃허브아이디.github.io/저장소이름/
```
